UI/UX Project "WatAV"

WatAV is an innovative smartwatch application designed to revolutionize how users navigate and explore their surroundings. Powered by Artificial Intelligence, WatAV assists users in discovering locations and accessing real-time map data and place details, all through a streamlined interface optimized for wearable technology.

Key Highlights:

  • AI-Driven Search: Implemented an intelligent search engine that uses AI to provide personalized and context-aware location recommendations.
  • Optimized Wearable Navigation: Designed a compact yet powerful map interface specifically tailored for small-screen smartwatch displays.
  • Instant Place Insights: Developed a seamless way for users to access detailed information about venues, such as ratings and operating hours, at a glance.
  • Accessibility & Mobility: Focused on hands-free interactions and responsive design to ensure ease of use while on the move.

Tools used: Figma

UMKMbid

UMKMBid is a dynamic investment marketplace built with Laravel and Tailwind CSS designed to bridge the gap between UMKM and potential investors. The platform features a sophisticated equity-based auction system, allowing business owners to secure capital by offering company shares to a network of competitive bidders.

Key Highlights:

  • Equity Auction Engine: Developed a real-time bidding system where UMKM can list funding requests with specific equity percentages and minimum investment thresholds.
  • Modern UI/UX Architecture: Leveraged Tailwind CSS to create a highly responsive, mobile-first interface, ensuring a seamless experience for users managing investments on the go.

Tools used: Laravel (PHP), Tailwind CSS, MySQL, Eloquent ORM, and MVC Architecture.

Klinik Serbabisa

Klinik Serba Bisa is a comprehensive, web-based clinic management system designed to streamline healthcare workflows. Built with Laravel 10 and the MVC architecture, the platform provides a robust solution for managing patient care, medical staffing, and clinical records through a highly organized and secure digital ecosystem.

Key Highlights:

  • Role-Based Access Control (RBAC): Implemented a secure permission system with dedicated dashboards and functionalities for Patients, Doctors, and Administrators.
  • Automated Clinical Management: Developed modules for real-time appointment scheduling, doctor availability tracking, and digital medical record keeping.
  • Optimized Data Architecture: Utilized Eloquent ORM and a relational database to ensure high-speed queries and data integrity across all medical modules.
  • Reporting & Documentation: Integrated PDF generation capabilities (via TCPDF/DomPDF) for instant creation of digital prescriptions, patient reports, and billing invoices.

Tools used: Laravel 10 (PHP), MySQL, Tailwind CSS, Blade Templating, and TCPDF/DomPDF.
